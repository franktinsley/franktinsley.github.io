CarTwitch Review Mode

⚠️ Important for App Store Reviewers:

This page provides a complete demo mode for testing all app functionality without requiring a Twitch account. Please follow the instructions below to access the full app experience.

Method 1 - Quick Activation (Recommended):

  1. Make sure you have installed CarTwitch on this device
  2. Tap the purple button below
  3. When prompted, select "Open" to launch CarTwitch
  4. The app will automatically enable review mode and simulate a logged-in account
Activate Review Mode

Method 2 - Manual Activation:

  1. Open the CarTwitch app
  2. Go to Settings
  3. Tap "Enable Review Mode" in the "App Store Review Mode" section
  4. Enter any username and password in the demo login screen
  5. The app will automatically simulate a logged-in account

What Review Mode Enables:

✓ Automatic simulated Twitch account login
✓ Following tab with auto-populated content
✓ Complete CarPlay integration
✓ Profile image in settings
✓ Full access to all subscription features

Exiting Review Mode:

  1. Go to Settings
  2. Tap "Exit Review Mode" in the "App Store Review Mode" section
  3. The app will return to its normal state without demo content

Note for Reviewers: This demo mode provides full access to all app features and functionality. No Twitch account is required for testing. If you encounter any issues, please contact us through App Store Connect.
